QuickExc Script Guide

Learn how to use and customize the QuickExc executor GUI for Roblox scripts.

Step 1: Download the Script

Save the QuickExc script as QuickExc.txt or QuickExc.lua and keep it ready to load in your Roblox executor.

Step 2: Open Roblox Executor

Open your Roblox script executor (Delta / Synapse / etc.), then use the "Load File" or "Execute Script" option to run QuickExc.

Step 3: Using QuickExc GUI

  • Drag the GUI anywhere on your screen.
  • Hover over buttons to see hover animation.
  • Click a button to execute a script. Buttons with destroy=true will close the GUI after execution.
  • Holding a button scales it up until released.
  • The flash effect will briefly show when executing a script.

Customization Guide

1️⃣ Adding New Buttons

Open the QuickExc script and find the scriptsList table. Each entry is a button:

{name = "Voidware Hub", url = "https://raw.githubusercontent.com/VapeVoidware/VW-Add/main/nightsintheforest.lua", destroy = true}

To add a new button, just add another entry with the same format:

{name = "New Script", url = "https://raw.githubusercontent.com/YourUsername/YourScript/main/newscript.lua", destroy = false}

2️⃣ Changing Button Sounds

Look for the click sound section:

local clickSound = Instance.new("Sound", mainGui)
clickSound.SoundId = "rbxassetid://3774415505"
clickSound.Volume = 0.6

Change SoundId to any Roblox audio asset you like.

3️⃣ Editing Titles / Guide Text

The top label is here:

local title = Instance.new("TextLabel")
title.Text = "Use buttons below to execute scripts:"

You can replace the text with any instructions, guides, or notes.

4️⃣ Changing Button Colors / Fonts

Each button is defined like this:

btn.BackgroundColor3 = Color3.fromRGB(40,40,40)
btn.TextColor3 = Color3.fromRGB(255,255,255)
btn.Font = Enum.Font.FredokaOne

You can change BackgroundColor3, TextColor3, and Font to customize appearance.

5️⃣ Destroy GUI Option

Buttons have a destroy property:

  • destroy = true → GUI closes after executing the script
  • destroy = false → GUI stays open

6️⃣ Scaling / Hold Effects

The GUI includes a scale animation on buttons:

  • Click → button quickly scales up then back.
  • Hold → button scales up and stays scaled until released.

Tips & Tricks

  • Always save a backup of your QuickExc script before modifying it.
  • You can add unlimited buttons — the scroll frame will adjust automatically.
  • Use meaningful names for buttons to easily identify scripts.
  • Change the flash color in the code for different visual effects.
